# Testing Documentation

This document describes the testing activities performed for the Cognito learning platform.

## Testing Type
Manual API testing was performed using Postman to verify backend functionality.

## Tested Features
- User Signup API
- User Login API
- Placement Test Questions Retrieval
- Placement Test Submission and Score Calculation
- User Data Retrieval for Dashboard

## Testing Process
Each API endpoint was tested using valid and invalid inputs to ensure correct responses and error handling.

## Results
All tested APIs responded correctly according to the system requirements.
